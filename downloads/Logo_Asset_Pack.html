Logo Usage Guidelines

Which Logo to Use

📱 Website & Digital
Use SVG format for scalability. Primary logo on light backgrounds, reversed on dark backgrounds.
📧 Email & Documents
Use PNG files (512px or 1024px). Stacked version works well in narrow spaces like email headers.
📱 App & Icons
Use favicon (ICO) or PNG 512px files. Primary solid logo works best for app icons.
🖨️ Print Materials
Use SVG or PDF files for best quality. Ensure minimum size of 40px for legibility.

Minimum Clear Space

Always maintain clear space around the logo. The minimum clear space is equal to the height of the letter "M" in the mark.

MD

Maintain minimum clear space on all sides

Sizing

✓ DO
• Use SVG for web (infinite scalability)
• Scale proportionally
• Minimum size: 40×40px
✗ DON'T
• Use sizes smaller than 40px
• Distort or stretch the logo
• Use rasterized (PNG) copies for print

Color Usage

✓ DO
• Use primary (dark) logo on light backgrounds
• Use white/reversed logo on dark backgrounds
• Use branded color variants intentionally
✗ DON'T
• Change logo colors arbitrarily
• Add gradients or effects
• Use logo on backgrounds with poor contrast

Installation & Implementation

For Website

<img src="logo.svg" alt="MD Build">

Always use SVG format for web for best scalability and file size.

For Email Signatures

Use the stacked logo variant (PNG 256px). Recommended dimensions: 120px wide, centered in email signature.

For Social Media

Use PNG 512px files. Recommended sizes:

  • Profile picture: 400×400px
  • Cover image: 1200×630px (scale logo proportionally)
  • Post graphics: 1080×1080px (Instagram)

For Print

Export from vector file (SVG) to PDF at desired size. Minimum print size: 0.5 inches (12.7mm) wide.

Technical Specifications

SVG Specifications

  • Color Space: RGB (sRGB for web, CMYK for print)
  • Vector Type: Pure vector (fully scalable)
  • Layers: Organized by color variant
  • File Size: Optimized for web (typically <5KB)

PNG Specifications

  • Color Space: RGB with transparency
  • Background: Transparent (32-bit PNG)
  • DPI: 72 DPI for screen, 300 DPI for print
  • Optimization: Compressed for web
